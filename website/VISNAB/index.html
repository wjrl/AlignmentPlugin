Description

The Visualization of Network Alignments using BioFabric (VISNAB) plugin is a software tool that is designed to give network researchers the ability to usefully visualize the results of a network alignment, even for very large networks. VISNAB provides the user with new insights into network alignment deficiencies that are not revealed through the simple evaluation and comparison of the standard numerical topological measures.

Significant features for VISNAB:
  • Node groups and link groups provide a flexible method to characterize an alignment's topology. With BioFabric V2, node and link group annotations clearly highlight the different groups.
  • The Group Layout allows users to visually assess common topological measures such as Edge Coverage (EC), Symmetric Substructure Score (S3), and Induced Conserved Structure (ICS).
  • The Cycle Layout allows users to find small and large scale misalignments when the perfect alignment is provided.
  • Our novel measures Jaccard Similarity (JS), Node Group Similarity (NGS), and Link Group Similarity (LGS) provide advantages over traditional methods, such as Node Correctness (NC), when comparing a generated alignment to the perfect alignment.
  • BioFabric provides the values of many common alignment measures, including EC, S3, ICS, and NC. It also includes our JS, NGS, and LGS.
  • Visually asses objective functions' performance on a pair of networks: BioFabric lets the user see how single measures and combinations of measures within objective functions affect the generated alignment's topology.
  • Lay correctly and incorrectly aligned nodes separately in the Group Layout. The user can choose the criterion for the correct alignment to be either Node Correctness or Jaccard Similarity (user can set the threshold value).

Download and Installation

Preparations

VISNAB is written as a BioFabric plugin, and requires that your first install BioFabric Version 2 (Beta), which is currently under development. Note that the current production Version 1.0 of BioFabric is not designed to support plugins; Version 2 is required. BioFabric is written in Java, and can be run on your Windows, Mac, or Linux computer. For Windows, you will first need to install the Java Runtime Environment (JRE), which can be downloaded from here (click the "Free Java Download" button). For the Mac, the executable already contains the JRE.

BioFabric Version 2 (Beta) is currently available either as a zip file for Windows (named BioFabric20B1.zip) or as a mountable disk image for Macs (named BioFabricInstallImageV2BetaRel1.dmg). These are both available from the BioFabric GitHub repository release page.

  • Windows: Unzip the BioFabric20B1.zip file and drag the BioFabric.exe file out of the archive and drop it on your desktop. Double-clicking on the desktop icon will start the program running. On Windows, you will probably need to agree to run software from an unknown publisher, or read a message that "Windows protected your PC", that the app is unrecognized, and you will need to click on the More info link to be able to choose the option to Run Anyway. (Note: The executable file has been signed, but the signing certificate has currently expired.)

  • Mac: Double-click the downloaded BioFabricInstallImageV2BetaRel1.dmg disk image file to mount it, and then open a Finder window for the mounted disk. From the Finder window, drag the BioFabric icon inside onto your desktop (or, if you prefer, into your Applications folder). Both the disk image and the BioFabric program itself have been signed. However, if Mac Gatekeeper has been set to restrict to only App Store, you will need to launch System Preferences, click on Security & Privacy, and on the General tab set Allow apps downloaded from: App Store and identified developers. If you take this step, you may wish to reset it back to App Store after starting BioFabric the first time.

Installing the Plugin

The VISNAB plugin is a Java JAR file. This compiled JAR file is available on the VISNAB GitHub repository release page. To install it, download the sVISNAB-V1.0.0.1.jar file, then simply drop that file into a directory on your computer. (Note that there should be only one copy of the VISNAB JAR file in that directory.) Start BioFabric, and use File->Set Plugin Directory... to point to the directory where you have placed sVISNAB-V1.0.0.1.jar. Restart BioFabric, and the VISNAB commands will appear in the Tools->Network Alignment menu.

Documentation

More complete documentation is being prepared.

To get started, once the plugin has been installed, VISNAB commands can be run from the main Tools->Network Alignment menu. VISNAB supports both SIF and LEDA (GW) graph file formats, and the ALIGN file format for alignment files. The graphs must be undirected.

Available commands include:

  • Edge Group Layout... will use the Default Node and Link Group Layout algorithm.
  • Orphan Edge Layout... will show only the Blue edges and the edges of nodes present in the Blue link group.
  • Alignment Cycle layout... will highlight misalignments of clusters and cliques.
  • Alignment Measures... will show common measures once an alignment has been loaded.
